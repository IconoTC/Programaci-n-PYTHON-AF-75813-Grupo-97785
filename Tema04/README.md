Laboratorios del Tema 04 del curso de Programación Python

Entorno Validado Python: Python 3.13

Es altamente recomendable que para usar estos laboratorios lo hagas 
sobre la VM Rocky Linux que te facilito con todo lo necesario preinstalado

Los contenidos de src y notebooks son los mismos laboratorios aunque puede 
haber alguna diferencia como evitar inputs en los notebooks para no interferir
la ejecución.

Se entregan en formato python nativo en el directorio src 
para que puedas usarlos en una terminal o en VS Code.

Los notebooks los puedes usar en VS Code en la Máquina Virtual 
que ya viene con la extensión preinstalada.
